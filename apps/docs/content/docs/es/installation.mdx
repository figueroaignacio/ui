## Instalación Global

<Steps>
  <Step>
    ### Instala el CLI de I7A UI globalmente para gestionar componentes en todos tus proyectos
    <Tabs defaultValue="pnpm">
      <TabsList>
        <TabsTrigger value="pnpm">pnpm</TabsTrigger>
      </TabsList>
      <TabsContent value="pnpm">
```cli
pnpm install -g i7a    
```
      </TabsContent>
    </Tabs>

  </Step>
  <Step>
    ###  Inicializa tu proyecto

    ```cli
    pnpm dlx i7a-ui init
    ```

  </Step>
  <Step>
   ### Agrega tu primer componente

      ```cli
    pnpm dlx i7a-ui add button
    ```

El código del componente se agregará directamente al directorio de componentes configurado, dándote control total para personalizarlo.

  </Step>
  
  <Step>
   ### Comienza a usar componentes
   Importa y usa los componentes en tu aplicación:

  <CodeBlockWrapper fileName="app/page.tsx" >
    ```tsx
    import { Button } from "@/components/ui/button"

    export function Page() {
      return (
        <Button>Click</Button>
      )
    }
    ```

  </CodeBlockWrapper>

  </Step>

</Steps>

## Instalación Manual

Si prefieres no usar el CLI, puedes instalar componentes manualmente:

<Steps>
  <Step>
    ### Instala Tailwind CSS

    Sigue la [guía de instalación de Tailwind CSS](https://tailwindcss.com/docs/installation) para tu framework.

  </Step>

    <Step>
    ### Crea el archivo de utilidades

    Crea un archivo `lib/utils.ts` con la utilidad `cn`:

    ```tsx
      import { clsx, type ClassValue } from 'clsx';
      import { twMerge } from 'tailwind-merge';

      export function cn(...inputs: ClassValue[]) {
        return twMerge(clsx(inputs));
      }
    ```

    Instala las dependencias requeridas:

    ```cli
      pnpm add clsx tailwind-merge
    ```

  </Step>

  <Step>
    ### Copia el código del componente

    Navega por la documentación de componentes, copia el código fuente y pégalo en tu directorio de componentes.

  </Step>
</Steps>
