## Comandos Principales

El CLI de I7A proporciona varios comandos para gestionar tus componentes:

### add

Agrega uno o más componentes a tu proyecto. El código fuente del componente se copiará a tu directorio de componentes.

```cli
pnpm dlx i7a-cli add button
```

### list

Muestra todos los componentes disponibles en el registro de I7A UI.

```cli
pnpm dlx i7a-cli list
```

### remove

Elimina componentes de tu proyecto.

```cli
pnpm dlx i7a-cli remove button
```

### update

Actualiza los componentes existentes a sus últimas versiones del registro.

```cli
pnpm dlx i7a-cli update button
```

Actualiza todos los componentes:

```cli
pnpm dlx i7a-cli update
```

### diff

Compara tu componente local con la versión del registro para ver qué ha cambiado.

```cli
pnpm dlx i7a-cli diff button
```
