## API Reference

### Spinner

| Prop        | Type                                                                                     | Default     | Description             |
| ----------- | ---------------------------------------------------------------------------------------- | ----------- | ----------------------- |
| `size`      | `'sm' \| 'md' \| 'lg' \| 'xl'`                                                           | `'md'`      | The size of the spinner |
| `variant`   | `'default' \| 'primary' \| 'muted' \| 'success' \| 'destructive' \| 'warning' \| 'info'` | `'default'` | Visual style            |
| `className` | `string`                                                                                 | -           | Additional CSS classes  |
