## API Reference

### Radio

| Prop              | Type                         | Default | Description                         |
| ----------------- | ---------------------------- | ------- | ----------------------------------- |
| `onCheckedChange` | `(checked: boolean) => void` | -       | Callback when checked state changes |
| `disabled`        | `boolean`                    | -       | Disables the radio                  |
| `name`            | `string`                     | -       | Name of the radio group             |
