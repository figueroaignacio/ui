You can override the default icon by passing the `icon` prop.

```tsx
<Callout icon={<MyCustomIcon />}>Callout with custom icon</Callout>
```

## API Reference

### Callout

| Prop        | Type                                                                | Default     | Description                                                                           |
| ----------- | ------------------------------------------------------------------- | ----------- | ------------------------------------------------------------------------------------- |
| `variant`   | `"default"` \| `"info"` \| `"warning"` \| `"danger"` \| `"success"` | `"default"` | The callout style.                                                                    |
| `icon`      | `React.ReactNode`                                                   | —           | Custom icon to display. If not provided, a default icon is used based on the variant. |
| `title`     | `string`                                                            | —           | The title of the callout.                                                             |
| `className` | `string`                                                            | —           | Additional CSS class names.                                                           |
| `children`  | `React.ReactNode`                                                   | —           | The callout content.                                                                  |
