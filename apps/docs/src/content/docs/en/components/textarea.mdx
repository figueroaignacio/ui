## API Reference

### Textarea

| Prop          | Type                              | Default     | Description                                           |
| ------------- | --------------------------------- | ----------- | ----------------------------------------------------- |
| `label`       | `string`                          | —           | Label text rendered above the textarea.               |
| `error`       | `string`                          | —           | Error message displayed below the textarea.           |
| `description` | `string`                          | —           | Helper text displayed between label and textarea.     |
| `size`        | `"sm"` \| `"default"` \| `"lg"`   | `"default"` | The size of the textarea.                             |
| `autoResize`  | `boolean`                         | `false`     | If `true`, textarea grows with its content.           |
| `disabled`    | `boolean`                         | —           | Disables the textarea.                                |
| `className`   | `string`                          | —           | Additional CSS class names.                           |
