## API Reference

### Progress

| Prop        | Type             | Default | Description                                              |
| ----------- | ---------------- | ------- | -------------------------------------------------------- |
| `value`     | `number \| null` | -       | The progress value (shows indeterminate if null/missing) |
| `max`       | `number`         | `100`   | The maximum value of the progress bar                    |
| `className` | `string`         | -       | Additional CSS classes                                   |
