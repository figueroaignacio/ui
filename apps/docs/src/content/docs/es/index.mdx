## Creá UIs. Mantené el Control.

Las librerías estándar funcionan hasta que necesitás comportamiento a medida. Ahí aparecen los wrappers, overrides de estilos y hacks.

NachUI soluciona esto. Te entregamos **el código fuente real**. Sos dueño de los componentes, tokens y patrones. Extendelos sin límites.

- **Propiedad Total**: El código vive en tu proyecto. Nada oculto.
- **Open Code**: Cada línea transparente, editable, auditable.
- **Diseño Unificado**: Componentes siguen reglas visuales estrictas.
- **Compound Components**: API semántica con dot-notation (`<Card.Header>`). Máxima flexibilidad.
- **Distribución CLI**: Instalá componentes con un comando. El código se copia directo a tu repo.
- **Nativo para IA**: Patrones estandarizados construidos para generación de código con IA.

## ¿Por qué Open Source?

Agregar un componente NachUI significa que el código fuente pasa a ser tuyo.

- Inspeccioná, auditá, modificá todo.
- Cambiá APIs o estructuras internas sin restricciones.
- Sin dependencias de terceros. Sin esperar PRs externos.

Trabajá directo con el código que necesitás.

## Compound Components

Evitá componentes monolíticos con infinitos flags booleanos. Componé tu UI usando elementos semánticos interconectados.

```tsx
// ❌ Monolítico (rígido, difícil de extender)
<Card title="Hola" description="Mundo" content="Este es el contenido" />

// ✅ Compound (flexible, legible)
<Card>
  <Card.Header>
    <Card.Title>Hola</Card.Title>
    <Card.Description>Mundo</Card.Description>
  </Card.Header>
  <Card.Content>
    <p>Este es el contenido</p>
  </Card.Content>
</Card>
```

**Beneficios:**

- **Cero Intrusiones**: Inyectá wrappers o elementos custom sin `renderProps`.
- **Jerarquía Clara**: El JSX coincide exacto con la estructura del DOM.
- **Tipado Enfocado**: Sub-componentes tienen props específicas. Autocompletado limpio en TypeScript.
- **Estado Implícito**: El padre comparte contexto con los hijos automáticamente.

## Construido para IA

- Código abierto y legible.
- Patrones repetibles. La IA los sigue sin ambigüedad.
- La IA extiende tu Design System sin romper la coherencia.

Construí más rápido. Mantené el control.
