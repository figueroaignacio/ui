## Referencia de API

### Radio

| Prop              | Tipo                         | Por defecto | Descripción                         |
| ----------------- | ---------------------------- | ----------- | ----------------------------------- |
| `onCheckedChange` | `(checked: boolean) => void` | -           | Handler cuando el estado cambia     |
| `disabled`        | `boolean`                    | -           | Evita interacción y reduce opacidad |
| `name`            | `string`                     | -           | Nombre del grupo de opciones        |
