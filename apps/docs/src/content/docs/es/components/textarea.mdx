## Referencia de API

### Textarea

| Prop          | Tipo                              | Por defecto | Descripción                                              |
| ------------- | --------------------------------- | ----------- | -------------------------------------------------------- |
| `label`       | `string`                          | —           | Texto del label renderizado arriba del textarea.         |
| `error`       | `string`                          | —           | Mensaje de error mostrado debajo del textarea.           |
| `description` | `string`                          | —           | Texto de ayuda entre el label y el textarea.             |
| `size`        | `"sm"` \| `"default"` \| `"lg"`   | `"default"` | El tamaño del textarea.                                  |
| `autoResize`  | `boolean`                         | `false`     | Si es `true`, el textarea crece con el contenido.        |
| `disabled`    | `boolean`                         | —           | Deshabilita el textarea.                                 |
| `className`   | `string`                          | —           | Clases CSS adicionales.                                  |
