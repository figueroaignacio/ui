## Referencia de API

### Spinner

| Prop        | Tipo                                                                                     | Por defecto | Descripción            |
| ----------- | ---------------------------------------------------------------------------------------- | ----------- | ---------------------- |
| `size`      | `'sm' \| 'md' \| 'lg' \| 'xl'`                                                           | `'md'`      | El tamaño del spinner  |
| `variant`   | `'default' \| 'primary' \| 'muted' \| 'success' \| 'destructive' \| 'warning' \| 'info'` | `'default'` | Estilo visual          |
| `className` | `string`                                                                                 | -           | Clases CSS adicionales |
