## Referencia de API

### Popover

| Prop           | Tipo                      | Por defecto | Descripción                                   |
| -------------- | ------------------------- | ----------- | --------------------------------------------- |
| `defaultOpen`  | `boolean`                 | `false`     | Estado inicial de apertura.                   |
| `open`         | `boolean`                 | —           | Estado controlado.                            |
| `onOpenChange` | `(open: boolean) => void` | —           | Callback cuando cambia el estado de apertura. |

### Popover.Trigger

| Prop      | Tipo      | Por defecto | Descripción           |
| --------- | --------- | ----------- | --------------------- |
| `asChild` | `boolean` | `false`     | Renderizar como hijo. |

### Popover.Content

| Prop         | Tipo                                           | Por defecto | Descripción                   |
| ------------ | ---------------------------------------------- | ----------- | ----------------------------- |
| `side`       | `"top"` \| `"bottom"` \| `"left"` \| `"right"` | `"bottom"`  | Posición relativa al trigger. |
| `sideOffset` | `number`                                       | `4`         | Distancia desde el trigger.   |
| `showClose`  | `boolean`                                      | `false`     | Muestra un botón de cerrar.   |
| `className`  | `string`                                       | —           | Clases CSS adicionales.       |

### Popover.Close

| Prop      | Tipo      | Por defecto | Descripción           |
| --------- | --------- | ----------- | --------------------- |
| `asChild` | `boolean` | `false`     | Renderizar como hijo. |
