Después de trabajar meses afinando patrones, componiendo interfaces y reutilizando ideas de mis propios proyectos… decidí convertir todo ese esfuerzo en algo más estructurado.

**Así nació I7A UI:**
un kit de componentes open source, inspirado en shadcn/ui, base-ui, Hero UI y construido con Tailwind + Framer Motion, pensado especialmente para proyectos en React y Next.js.

No es una librería.
No es un paquete de npm.
No es magia.

Es código real, para copiar, pegar y ser dueño de tu capa de componentes.

## ¿Qué hace diferente a I7A UI?

### Propiedad Directa

Acá no instalás un paquete ni dependés de cajas negras.
Cada componente vive dentro de tu proyecto.
Lo ves, lo tocás, lo modificás.
Si algo no te gusta, simplemente lo editás.

### Diseño Unificado

Todos los componentes siguen los mismos patrones:
APIs consistentes, estilos coherentes y comportamiento predecible.

Nada de mezclar soluciones que no combinan.

### Copiar, pegar y listo

Muchos componentes utilizan las herramientas que usamos todos los días:

- clsx
- class-variance-authority
- tailwind-merge

Para que integrar sea fácil, rápido y natural.

### Distribución simple

Solo agregás el componente y ya forma parte de tu proyecto, con una estructura pensada para escalar.

### Animaciones con Framer Motion

Las animaciones están integradas con un estilo elegante, cuidando performance y estética.

### Un proyecto Indie

I7A UI no intenta ser “la nueva gran librería”.
Es un proyecto indie, abierto, honesto, y hecho con amor al frontend.

### Nativo para IA

El código es claro, estructurado y consistente, ideal para que herramientas de IA:

- generen variantes,
- propongan mejoras,
- o creen componentes nuevos siguiendo tu sistema.

## ¿Por qué lo hice?

Porque estaba cansado de reinventar botones, modales y layouts en cada proyecto.
Porque quería velocidad, consistencia y un estilo propio.
Y porque prefiero ser dueño del código que dependo de un paquete que vive en npm.

I7A UI es mi forma de crear un sistema de diseño real, flexible y totalmente abierto.

Si les interesa ver el progreso, dar feedback o colaborar, más que bienvenidos. Esto recién empieza. ✨
